# Sorting Algorithms Python
All generic sorting algorithms in Python 3.x Standard Library

> ## Bubble Sort 
>> Requirement:  
>> Python 3.x Standard Library
>
>> Time Complexity : O(n<sup>2</sup>)  
>> Space Complexity: O(n)

> ## Insertion Sort 
>> Requirement:  
>> Python 3.x Standard Library
>
>> Time Complexity : O(n<sup>2</sup>)  
>> Space Complexity: O(n)

> ## Selection Sort 
>> Requirement:  
>> Python 3.x Standard Library
>
>> Time Complexity : O(n<sup>2</sup>)  
>> Space Complexity: O(n)
